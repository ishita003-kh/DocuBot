# DocuBot — AI Document Q&A System
A RAG-based chatbot that answers questions from your PDF documents.

## Built With
Python, LangChain, FAISS, Streamlit, Groq (Llama 3)

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. streamlit run index.py

## Features
- Upload any PDF
- Ask questions in natural language
- Answers grounded in document context
